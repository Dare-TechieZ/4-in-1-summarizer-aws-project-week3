# 4-in-1 AI Summarizer

An all-in-one AI-powered productivity tool designed to streamline information processing. This application provides concise, actionable insights for long meeting transcripts, complex articles, technical codebases, and general text.

---

## 🚀 Live Demo

**Live Application:** https://4-in-1-summarizer-aws-project-week3.vercel.app/

---

## ✨ What I Built

### Multi-Mode Processing

Built-in logic tailored for four specific content types:

- 📋 **Meetings**
  - Extracts action items, decisions, and discussion points.

- 📰 **Articles**
  - Creates concise overviews and highlights key concepts.

- 💻 **Code**
  - Generates technical explanations and logic walkthroughs.

- 🌐 **General / Universal**
  - A versatile mode for summarizing any other text input.

### 📝 Professional Formatting

Outputs are structured using **Markdown** to ensure they are clean, readable, and ready for documentation.

### 🎨 Polished UI

Integrated **Tailwind CSS Typography Plugin** to provide a distraction-free, professional reading experience.

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/Dare-TechieZ/4-in-1-summarizer-aws-project-week3.git
cd 4-in-1-summarizer-aws-project-week3
```

### 2. Install Dependencies

```bash
npm install
```

or

```bash
pip install -r requirements.txt
```

### 3. Configure API Keys

Create a `.env.local` file in the project root and add:

```env
GOOGLE_API_KEY=your_gemini_api_key_here
GROQ_API_KEY=your_groq_api_key_here
```

### 4. Run the Application

```bash
npm run dev
```
